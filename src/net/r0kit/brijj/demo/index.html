Welcome to BriJJ

Brijj is an Ajax remoting framework for Java to make it easy for web pages to interact with Java classes.

Inside this Demo

The generated configuration / test pages show how BriJJ is setup.

Documented examples of how BriJJ works:

  • Getting Started with BriJJ:
    • First steps: Start with updating some simple text and then including one page in another
    • More visual: Editable Table allows a user to edit a table of data, or you can search similar data
  • Binary file handling:
    • File upload is easy - this example shows image manipulation
    • Download is just as easy - this time with custom PDF generation

BriJJ is testing installation ...

BriJJ uses JavaScript to make some initial checks. If you can see this message then either you are a very fast reader, or JavaScript is not working properly in your browser.

Since AJAX requires JavaScript, you will need to enable JavaScript in your browser before you can test BriJJ. Some very old browsers like IE 4.0 have very out of date support for JavaScript and are currently unsupported. Please upgrade your browser.

BriJJ is testing installation ...

This is a fairly advanced page that tests BriJJ. If you can see this message for a prolonged period of time then you should check the following:

  • Can you see the test pages?
    If not it is likely that BriJJ has not deployed in your web server properly; Check the web server console and log files for error messages.
  • If you can see the test pages then BriJJ is loaded but is something is preventing BriJJ working. Check your browser JavaScript console for error messages, or see the main BriJJ website for more information.

If none of these steps fixes the error, please report it to the BriJJ mailing list, including your browser, server, OS configuration and any error messages that appeared either in the browser (including the JavaScript console if one exists) and the server console log.
