Welcome to BriJJ

Brijj (Browser Remote Interface from Javascript to Java) is an Ajax remoting framework for Java to make it easy for web pages to interact with server-side Java classes.

The generated test pages let you experiment with the remote methods defined for this installation.
Working examples of BriJJ:
  • Getting Started (Hello, World).
    • Manipulating strings
    • Editable Table allows a user to edit a table of data
    • Search a table of data
  • Uploads and Downloads
    • Downloading a file - in this case with custom PDF generation
